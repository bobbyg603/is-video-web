# is-video

> Check if a filepath is a video file in a browser compatible fashion.


## Install

```
$ npm install --save is-video-web
```


## Usage

```js
var isVideo = require('is-video-web');

isVideo('src/unicorn.mp4');
//=> true

isVideo('src/unicorn.txt');
//=> false
```
